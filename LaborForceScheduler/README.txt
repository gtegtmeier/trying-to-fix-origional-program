LaborForceScheduler v3 NEW — Portable Package

1) Unzip this folder anywhere (Desktop is fine).
2) Double-click Run_Scheduler.bat to open the app.
3) Optional: Build_EXE.bat builds a standalone EXE in this folder.
4) Optional: Create_Desktop_Shortcut.bat creates a Desktop shortcut.

Folder layout:
  LaborForceScheduler_V3_New/
    LaborForceScheduler.py
    Run_Scheduler.bat
    Build_EXE.bat
    Create_Desktop_Shortcut.bat
    Create_Desktop_Shortcut.vbs
    data/
    history/
    exports/
    assets/

Data is stored in ./data/scheduler_data.json (relative path) by default.
